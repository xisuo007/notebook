4. 代码    ctrl + shift + `
5. 超链接    ctrl + K
6. 图像    ctrl + shift + I
7. 引用    ctrl + shift + Q
8. 有序/无序列表    ctrl + shift + [ / ]
9. 复制为MD    ctrl + shift + C
10. 粘贴为纯文本    ctrl + shift + V
11. 源代码模式    ctrl + /
12. 大纲视图    ctrl + shift + 1
13. 文件视图    ctrl + shift + 2
14. 文件树视图    ctrl + shift + 3
15. 全屏切换    F11
